# VaultAI-CP

# 1. Open the project in Android Studio Ladybug (2024.2+) or newer

# 2. Sync Gradle — it will download all dependencies

# 3. Enable Developer Options + USB Debugging on your Android device

# 4. Run on device (NOT emulator — USB Host API requires real hardware)
#    Run > Run 'app'

# 5. Plug in a USB OTG drive containing a model file (e.g., model.gguf)
#    → The app auto-detects it and shows the permission dialog
